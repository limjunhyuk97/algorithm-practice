## 결국 구해야할 것이 무엇인가? / 이전 단계의 연산결과가 새 연산에 적용이 되는가?

### BOJ_12865. 평범한 배낭
  - 전체 무게가 ~ 일때 -> **결국 구해야하는 것 : 최대로 담을 수 있는 보석의 가치**
  - 보석 무게 , 보석 갯수 -> **보석 가치**
  - 이전 단계의 수치를 이용 (max 가치)

### BOJ_9084. 동전
  - 전체 금액이 ~ 일때 -> **결국 구해야하는 것 : 동전 조합의 수**
  - 금액, 동전 종류 -> **조합의 수**
  - 이전 단계의 수치를 이용 (+= 조합의 수)
